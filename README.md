# 🚀 AI Measure Pro YOLO V5

AI-based real-time object measurement system using computer vision.

## 🔥 Features

- **YOLOv8 Object Detection**: High-speed real-time object identification.
- **MediaPipe Person Measurement**: Accurate human pose and dimension tracking.
- **Auto Calibration**: Smart calibration using a standard Bank Card.
- **AI Explanation Panel**: Detailed insights into detection results.
- **SOC Logging System**: Security operations center style logging for audits.
- **TapMap Visualization**: Interactive heatmaps and data plotting.
- **Voice Announcements**: Real-time audio feedback for measurements.
- **Barcode Scanner**: Integrated scanning for product information.
- **PDF Export**: Generate professional measurement reports.
