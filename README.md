# 🚀 AI Measure Pro YOLO V5

AI-based real-time object measurement system using computer vision.

## 🔥 Features
- YOLOv8 Object Detection
- MediaPipe Person Measurement
- Auto Calibration (Bank Card)
- AI Explanation Panel
- SOC Logging System
- TapMap Visualization
- Voice Announcements
- Barcode Scanner
- PDF Export

## ▶ Run
pip install -r requirements.txt  
python app.py
